# Implementation Qualification

Loaded when the classifier labels an opportunity `implementation`. Your job: qualify the
deal against the firm's ICP, read the deal shape, and compile the package the **delivery /
proposal team** needs to start a response.

## When this applies
Net-new build: new system selection and stand-up, greenfield platform, replacing a legacy
system with a new one. Distinguish from *optimization* (improving a system already in prod)
and *pre-implementation* (advisory before any build is scoped).
