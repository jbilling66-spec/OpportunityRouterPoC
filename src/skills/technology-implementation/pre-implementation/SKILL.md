# Pre-Implementation Planning Qualification

Loaded when the classifier labels an opportunity `pre_implementation`. Qualify, read the
advisory deal shape, and compile what the **advisory team** needs — and flag the downstream
implementation pull-through, since this is often the front door to a larger build.

## When this applies
Early-stage advisory: roadmaps, readiness/maturity assessments, system selection, business
cases, current-state analysis. No system selected, no build scoped yet.
